Módulo vacaciones

Inicio

Desde este módulo el empleado con acceso al sistema podrá realizar la solicitud de sus vacaciones, y el personal encargado de RRHH podrá autorizar o no las vacaciones.

Cabe mencionar que el empleado que no es del departamento de RRHH y que no tiene los permisos necesarios para autorizar vacaciones, no podrá visualizar la opción de "Autorizar vacaciones"

Solicitar vacaciones

Solicitante: Acá se mostrará el nombre de la persona que esta solicitando las vacaciones, pero si el de RRHH esta gestionando las vacaciones, le aparecerá el listado de los empleados.

Autoriza: Se mostrará un listado con las personas que tienen la autoridad para aceptar las vacaciones.

Tipo: Es el motivo por el cual toma las vacaciones

Fecha de incio: La fecha en la cual se tomarán las vacaciones

Fecha de finalización: Cuando será el último día que gozará de vacaciones el empleado

Descripción: Espacio para anotar algo con referencia a este periodo de vacaciones solcitadas.

Formulario

Autorizar vacaciones

Panel de autorización: Se muestran las solicitudes gestionadas por los empleados o personal de RRHH. Al posicionarse sobre la solicitud podrá dar un clic derecho y se le mostrará un menu contextual dónde podrá elegir "Autorizar" o "Rechazar" las vacaciones solicitadas.

Solicitar permiso

Se realizó la actualización del apartado para solicitar permisos, este incluye el tipo de permiso a solicita, la fecha y un paratado para describir el porque del permiso.

Actualización

Se agrego un espacio para poder realizar la carta de solicitud de vaciones, acá se pued dejar configurado el texto para que el solicitante solo cambie los datos de fecha, nombre y el tiempo, pero la estructura se puede dejar de un vez creada.

Esta forma de realizar la carta a futuro representa una disminución es costos bastante significativa, ya que en lugar de almacenar archivos de 5MB por cada solicitud, solo se estan guardando algunos kilobits, por solo ser texto.

Autorizar vacaciones

En este apartado tendrá a la vista todas las solicitudes realizadas por los empleados para tomar sus vaciones y dando clic derecho soble el solicitante podrá obtener un menu contextual que le mostrará las opciones para realizar con dicha petición.

La opción "Ver", le mostrará la información de la solicitud. La opción "Autorizar", le enviará una notificación al solicitante para tomar sus vaciones. La opción "Rechazar", negará la solicitud.

Carta de solicitud

Al dar clic en la opción "Ver", en solicitudes de vacaciones, se mostrará la carta que realizo el solicitante.
