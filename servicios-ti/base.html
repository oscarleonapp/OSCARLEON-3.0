Módulos del sistema

Empleados

Puede gestionar a sus empleados para la entrega de prodcutos.

Clientes

Vea que clientes son los que más le compran, también podrá gestionar cuales son los productos con mejor movimiento.

Notificationes

Cada transacción envia una notificación al administrador del sistema.

Pagos

Puede configurar pagos con Transferencias, Depositos o Pago contra entrega.

POS Venta

Le permite un punto de venta para gestionar salidas de productos de una manera más eficiente, puede utlizar códigos de barra y vender desde una tablet, tal como lo hace cualquier comercio físico.
