Sistema para Inventarios - InvenPRO v2

InventPRO v2

Permite tener el control del inventarios de los productos, puede gestionar los pedidos, crear facturas, tener un catálogo con categorias, atributos, caracteristicas, marcas, proveedores y un fácil manejo del stock.

Empleados

Puede gestionar a sus empleados para la entrega de prodcutos.

Clientes

Vea que clientes son los que más le compran, también podrá gestionar cuales son los productos con mejor movimiento.

Pago

Puede aceptar pagos con Transferencias, Depositos o Pago contra entrega.

RockPOS

Le permite un punto de venta para gestionar salidas de productos de una manera más eficiente, puede utlizar códigos de barra y vender desde una tablet.
